# EcoEarn

## Overview

EcoEarn is a mobile application developed to encourage sustainable recycling habits through a reward-based system. The application allows users to record recycling activities, track their progress, and earn rewards for contributing to environmental sustainability.

This project was developed as part of the Mobile Application Development course and aligns with **Sustainable Development Goal (SDG) 12: Responsible Consumption and Production**.

---

## Problem Statement

Many individuals, especially students, lack motivation to participate in recycling activities due to the absence of a convenient tracking and reward system. EcoEarn addresses this issue by providing a digital platform that encourages environmentally responsible behaviour through incentives and activity monitoring.

---

## Objectives

* Promote recycling awareness and participation.
* Encourage sustainable environmental practices.
* Provide a simple platform to track recycling activities.
* Reward users for their contributions towards sustainability.
