Project 1 – EcoEarn MVP

Challenge & Sustainable Solution

Low recycling engagement due to lack of immediate rewards. EcoEarn (MVP) introduces a point-based ecosystem: users log recyclable waste → earn points → redeem rewards (vouchers, eco-products). Direct impact on SDG 12.

GitHub Repository (Project 1)

Technical Takeaways

Designed end-to-end user flow: points ledger, reward catalog, shared preferences for session. Improved debugging, version control, and user-centered design. Architected the first iteration of EcoEarn using MVP pattern.
