Project 1 – EcoEarn (MVP)

Problem Statement & Solution

Communities lack accessible incentives to recycle — EcoEarn solves this by a points-based system: users earn credits for verified recycling actions, then exchange them for rewards (coupons, eco-products). This increases recycling rates and fosters responsible production habits.

GitHub Repo VSR Demo

Key Takeaways

Designed the complete architecture: UI flows, shared preferences for session, and recycler view adapters. Understood the importance of user-centered design and basic data persistence. Strengthened my Git collaboration and debugging discipline.

Project 2 – EcoEarn Enhanced

Advanced Ecosystem

EcoEarn v2 transforms into a feature-rich app with real-time cloud sync, sensor-based activity tracking, and external recycling center APIs to empower users with actionable data.

  • Room Database – offline storage for transactions and user points
  • Firebase Auth & Firestore – multi-device sync & secure login
  • REST API (Retrofit) – fetch nearby recycling/drop-off points
  • Sensor Integration – step counter / accelerometer to earn “green points”
  • MVVM + LiveData – maintainable and testable architecture
