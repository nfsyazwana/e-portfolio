Project 1 – EcoEarn (MVP)

Problem Statement

Low recycling participation due to lack of guidance and incentives. People want to recycle but don't know how to separate waste correctly, and there's no reward for their effort.

Working Features

  • User registration & login (SharedPreferences)
  • Item scanner with category selection (Plastic/Glass/Paper/Metal)
  • Points ledger and reward catalog
  • Form submission + photo proof upload
  • Cash-out to e-wallet
GitHub Repository YouTube VSR

Lessons Learned

Built complete user flow from scratch. Strengthened understanding of Android architecture, data persistence, and user-centered design. Learned to debug real-world issues and implement reward logic efficiently.

Project 2 – EcoEarn Enhanced

Problem Statement & Advanced Features

Extending EcoEarn into a production-grade platform with real-time sync, sensor gamification, and API integration for nearby recycling centers.

  • Room Database – offline transactions & points history
  • Firebase Auth & Firestore – cloud sync & secure login
  • REST API (Retrofit) – live recycling center locator
  • Sensor Integration – step counter to earn green points
  • MVVM + Coroutines + Hilt – clean architecture
  • Google Maps SDK – nearest eco centers
GitHub Repository YouTube VSR

Lessons Learned

Mastered backend integration with Firebase, handled authentication tokens, managed asynchronous data streams, and created a scalable reward engine. Implemented clean architecture with repository pattern and unit tests.

Reflection & Learning Journey

Starting from basic UI layouts in Lab 1 to building a fully integrated, cloud-connected, sensor-driven application in Project 2 has been transformative. Each lab built a strong foundation — UI design, event handling, Material Design, navigation, and local persistence. Project 1 challenged me to think like a product owner, solving real user pain points (confusion about recycling + lack of rewards). Project 2 pushed me into advanced territory: Firebase sync, REST APIs, step sensors, and production-ready architecture (MVVM, Hilt, Coroutines).


Challenges overcome: Debugging Room migrations, handling API rate limits, and implementing sensor-based gamification. Each obstacle taught me resilience and deeper Android internals.


SDG 12 Impact: EcoEarn directly addresses responsible consumption by making recycling easy (AI guidance) and rewarding (cash incentives). My work contributes to reducing landfill waste and promoting circular economy habits.
